Firebase Realtime Database — collab en temps réel
Étape 1 — Créer le projet Firebase
  1. Va sur console.firebase.google.com
  2. Créer un projet → nommer ava2i-roadmap
  3. Realtime Database → Créer → Mode test
  4. Paramètres → App Web → copier le firebaseConfig
Étape 2 — Coller dans le fichier HTML
  1. Ouvrir index.html dans un éditeur
  2. Chercher FIREBASE_CONFIG
  3. Remplacer les VOTRE_... par tes vraies valeurs
  4. Sauvegarder et uploader sur GitHub Pages
💡 Gratuit — Firebase offre 1 Go de stockage. Chaque modification est visible immédiatement pour tous les collaborateurs.
